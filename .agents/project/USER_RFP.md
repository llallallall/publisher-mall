1. 프로젝트 개요
출판사에서 모의고사 상품을 온라인으로 판매하고, 자료실 및 후기 관리 기능을 포함한 웹사이트를 구축합니다.

2. 프로젝트 목적
모의고사 책 온라인 판매
자료실을 통한 부가 자료 제공
구매자들의 후기 및 평가 관리

3. 개발 범위
턴키 개발 (기획/디자인/개발/퍼블리싱) 
쇼핑몰 솔루션 사용 가능 
- 쇼핑몰 웹사이트 (모바일 반응형 포함) 
- 관리자 페이지 

4. 필요 기능 
회원 페이지 
- 회원가입/로그인
- 상품 리스트/상세/구매
- 결제 시스템 연동(국내 PG사)
- 후기 작성 및 관리
- 자료실 파일 다운로드 
관리자 페이지 
- 상품/주문/회원 조회 및 관리
- 자료실 관리: 무료 자료, 모의고사 등급컷 등 파일을 게시하고, 고객이 다운받을 수 있음 
- 후기 관리: 다른 플랫폼의 후기를 쇼핑몰로 가져올 수 있음 

5. 운영/유지보수 범위
런칭 후 6개월 하자보수 포함
