# Design System: Academic Authority (Solid Serenity)

## 1. 디자인 컨셉
- **Solid Serenity**: 출판사의 학문적 권위(Solid)와 학습자의 평온한 환경(Serenity)의 조화

## 2. 시각적 가이드라인
- **Colors**:
  - `Primary`: `#0F172A` (Trust Navy) - 신뢰와 깊이감
  - `Accent`: `#EAB308` (Academic Gold) - 성취와 집중
- **Typography**: 헤드라인 `font-black`, `tracking-tighter` (-0.04em) 적용
- **Elements**: 
  - **Basalt Curve**: 버튼과 카드에 `rounded-2xl` 이상의 곡률 적용
  - **Blueprint Grid**: 배경에 미세 격자(5%)를 배치하여 정밀 설계된 문항 이미지 강조

## 3. 인터랙션 표준
- **Soft Skeleton**: 모든 데이터 로딩 시 부드러운 애니메이션 적용
- **Micro-interactions**: 버튼 호버 시 정교하고 부드러운 스케일 변화 제공
