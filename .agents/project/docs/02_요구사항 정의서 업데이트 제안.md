# **\[Software Requirements Specification\] 출판사 모의고사 통합 커머스 플랫폼**

## **1\. 프로젝트 개요 (Project Overview)**

* **프로젝트명**: 출판사 모의고사 통합 커머스 및 학습 지원 플랫폼  
* **핵심 목표**: **Academic Authority**(학문적 권위)를 담은 UI와 안정적인 결제/자료 제공 시스템 구축  
* **기술 표준**: Nuxt 4 (compatibilityVersion: 4), Node.js (Nitro), MySQL, Supabase Auth

**3\. 비기능적 요구사항 (Non-functional Requirements)**

### **3.1 성능 및 안정성 (Performance)**

* **초고속 렌더링**: Nuxt 4의 하이브리드 렌더링을 활용하여 모바일 페이지 진입 속도 최적화  
* **이미지 최적화**: 모든 도서 썸네일은 NuxtImg를 통해 **WebP**로 변환하여 페이로드 절감

### **3.2 보안 및 법적 준수 (Security & Legal)**

* **데이터 격리(RLS)**: 사용자가 타인의 주문/개인정보에 절대 접근할 수 없도록 DB 레벨에서 차단  
* **개인정보 보호**: 결제 관련 민감 정보는 서버에 저장하지 않고 PG사 보안 토큰만 사용  
* **다운로드 보안**: 자료실 링크에 유효기간이 포함된 **Signed URL** 적용

### **3.3 UI/UX 표준 (Design Standard)**

* **Basalt Curve**: 학습자에게 정서적 안정을 주는 rounded-3xl 곡률 시스템 적용  
* **Blueprint Grid**: 배경에 미세 격자(5% 농도)를 배치하여 학문적 정밀함과 신뢰감 시각화  
* **Soft Skeleton**: 데이터 호출 시 부드러운 애니메이션(bg-gray-100/50)으로 고품질 인상 부여
